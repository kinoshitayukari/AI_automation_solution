# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1x-6pFx3T06P9AZzoPz6o1wGdwvRZZx8b

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `VITE_GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key. The blog admin screen can also accept the key directly and now runs a 4ステップの生成パイプライン (Gemini 2.0 Flash → Gemini 2.5 Flash) to research 10日本語サイト、統合アウトライン、Markdown本文、HTMLを順に作成します。
3. Run the app:
   `npm run dev`
